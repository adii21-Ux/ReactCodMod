Setup Guide for ReactCodMod

1. Clone the Repository

git clone https://github.com/adii21-Ux/ReactCodMod.git

2. Install Dependencies

cd ReactCodMod
npm install

3. Install Live Server Extension for VS Code

Follow these steps to install the Live Server extension:

  1. Open VS Code
  2. Go to the Extensions view by clicking the Extensions icon in the Sidebar or pressing Ctrl+Shift+X
  3. Search for Live Server
  4. Click Install on the Live Server extension

4. Run the Server

npm start

This command will start the development server.

5. Run the Frontend

Open homepage.html in VS Code and click on Go Live at the bottom-right corner of the window.

This will start the live server and open the frontend in your default web browser.

6. Explore

Now you can explore the application, make changes, and see them reflected live in your browser.
